## Recriando a pagina inicial do Instagran

Neste desafio foi proposto recriar a pagina inicial do instagram a partir das aulas de HTML e CSS, com o objetivo de fir os seguintes aprendizados:

 - Flex container
 - Flex box
 - Justify content
 - Align Items
 - Align Self
 - Flex Wrap
 - Flex Shirink
 - Flex Grow
 - Flex Flow
 - Flex Basis
 - Order
